🚨
Life-Limited Parts require back-to-birth traceability per 14 CFR 43.10. Parts at or beyond life limit must be scrapped immediately — no exceptions, no extensions. Every removal from aircraft must update TSN/CSN here.
ℹ️
Expendable parts are tracked by lot/batch number and expiry date — not individual serial numbers. Use the Batch Recall tool to quarantine an entire lot instantly.
⛔
Quarantined parts cannot be installed on any aircraft. They must be physically segregated from serviceable stock and clearly marked. Disposition required before any return to service.

1
Create Firebase Project
Go to console.firebase.google.com → Add project → name it "airways-mro"
2
Add Web App
Project Settings → Add app → Web → Copy the firebaseConfig object
3
Enable Email/Password Auth
Authentication → Sign-in method → Email/Password → Enable
4
Create Firestore Database
Firestore Database → Create database → Start in test mode
5
Paste Config Below → Save
Fill in each field from your firebaseConfig object and click Save & Test
